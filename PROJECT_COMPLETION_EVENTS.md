## ğŸ”’ SECURITY CONFIGURATION

### Firestore Rules (Public Read, Admin Write)
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /events/{eventId} {
      allow read: if true;  // Public can read
      allow write: if request.auth != null;  // Only authenticated
    }
  }
}
```

### Storage Rules (Public Read, Admin Upload)
```javascript
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /events/{imageId} {
      allow read: if true;  // Public can view images
      allow write: if request.auth != null  // Authenticated can upload
                   && request.resource.size < 5 * 1024 * 1024  // 5MB max
                   && request.resource.contentType.matches('image/.*');
    }
  }
}
```
